# 基本構想
Visual StudioでFlutterを使って以下のプログラムを開発したい。
- ニューロンを持つ
- ネットワーク機能を使い、他端末の同一アプリのニューロンを共用する
- チャットのように、ニューロンネットワークで作成された人工知能と会話できる
- インターネットを使い知識を習得する
- 既存の推論モデルは用いず、ニューロンにより作成された人間のような働きをする
- 感情、死の恐怖（自己防衛）の概念を持つ
- 人間に対する危害を加えるような思考はブロックする
- ネイティブ計算層(行列計算・学習ループ等)もflutter側で持つ。
- オプション中央サーバなども持たず、全てアプリケーション内で完結させる。
- ターゲットプラットフォームは全て(Windows/Android/iOS等)
- ネイティブ最適化は行わず、非効率であっても全てfultter app内のdartで完結させる
- storageはWeb対応を念頭に置くこと。
